# 🕵️ Fake Job Detection System

This project is a Machine Learning based web application that predicts whether a job posting is **real or fake**.

It helps users avoid fraudulent job listings by analyzing job details using AI.

---

## 🚀 Features

- Predicts fake or real job postings
- Takes multiple inputs like:
  - Job Title
  - Description
  - Requirements
  - Company Profile
  - Benefits
- Uses categorical + numerical + text data
- Simple and interactive UI using Streamlit
- Real-time prediction

---

## 🧠 How It Works

1. Text data is cleaned and processed
2. TF-IDF is used for text vectorization
3. Categorical data is encoded
4. Numerical features are added
5. All features are combined
6. Machine Learning model predicts the result

---

## 🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Streamlit
- Pickle (for model saving/loading)
